Определяет стиль проходов обработки для лучшего контроля.

Спираль (Spiral) - Обрабатывает деталь непрерывным резанием по длине детали. Для обеспечения чистоты поверхности на двух концах вырезается полный круг.
Линия (Line) - Перемещает инструмент вдоль поворотной оси по прямым линиям, при этом поворотная ось используется только в конце каждого прохода для repositioning детали.
По окружности (Circular) - Вращает деталь при фиксированном положении инструмента, фактически обрабатывая окружность. Затем инструмент смещается на требуемую величину и обрабатывает следующую окружность.

Линия
