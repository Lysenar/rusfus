Ограничивает положение оси X относительно пределов радиусов или точки контакта радиуса вершины инструмента.

Темно-синяя линия представляет внутренний предельный радиус.

Режущая кромка устанавливает жесткое ограничение для положения X реза. Этот предел определяется Внутренним радиусом для наружной обработки или Внешним радиусом для внутренней обработки.

Ограничение по режущей кромке

Точка контакта позиционирует радиус вершины инструмента по касательной точке реза. Больший радиус вершины создаст большее перекрытие.
