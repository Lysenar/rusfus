Тип соединения.

  • Полоса: Непрерывная лента, соединяющаяся с деталью по всей верхней кромке
  • Трапеция: Треугольники с усеченными вершинами
  • Точки разрыва: Как трапеция, но с дополнительным, перевернутым рядом трапеций сверху, формирующих X-образные соединения
  • Треугольники: Треугольники, которые просто заканчиваются острыми вершинами, для очень деликатного крепления поддержек

Типы соединений
Слева: Трапеции, точки разрыва, полоса. Четвертый тип, треугольники, не показан, так как это по сути трапеции без усечения.
