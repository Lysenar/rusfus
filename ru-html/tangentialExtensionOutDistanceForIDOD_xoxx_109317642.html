Удлиняет профиль геометрии на Заднем Пределе по касательной на заданное расстояние. В результате траектория также удлиняется на Заднем Пределе по касательной.

Полезно для удлинения выхода инструмента из канавки, который выполняется на рабочей подаче.

Касательное удлинение = 0мм

Касательное удлинение = 5мм

Примечание: Убедитесь, что вокруг Заднего Предела достаточно места для безопасного выхода инструмента.
