Управляет подачей инструмента при отводе — движении инструмента при выходе из заготовки после реза.

Для сокращения времени цикла рекомендуется устанавливать подачу отвода равной режущей подаче. Подачу отвода можно увеличить при использовании очень низкой режущей подачи, чтобы инструмент быстрее выходил после завершения реза.
