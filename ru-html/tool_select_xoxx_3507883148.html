Открывает диалоговое окно Библиотеки инструментов для выбора существующего инструмента или создания нового.

Для этой операции наиболее подходят инструменты для бокового резания, включая:

  • Торцевую фрезу
  • Коническую фрезу
  • Фрезу с радиусом
  • Сферическую и шаровую фрезу
  • Фрезу для фасок
