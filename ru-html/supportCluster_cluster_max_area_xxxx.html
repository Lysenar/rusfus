Задает максимальный размер обнаруживаемой области. Если размер области превышает указанное значение, поддержки не генерируются.

Значение должно быть не менее 0.1 см². Эта настройка полезна, если нужно генерировать другой тип поддержек для больших областей. Например, для областей меньше заданной Максимальной площади можно использовать стержневые поддержки, а для всего остального — объемные.
