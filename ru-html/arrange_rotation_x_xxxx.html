Задает шаг приращения, на который компоненты могут поворачиваться вдоль оси X.

  • С шагом 180 град: Поворот с шагом 180 градусов.
  • С шагом 90 град: Поворот с шагом 90 градусов.
  • Пользовательский: Поворот с пользовательским шагом в градусах.
  • Нет: Без поворота вдоль оси X.

Подсказка: Эта глобальная настройка может быть переопределена для отдельных компонентов в столбце Поворот таблицы компонентов.
