Открывает библиотеку инструментов для выбора существующего или создания нового инструмента.

Инструменты для этой операции:

  • Сегментные: бочкообразные, линзовые, овальные, конические.
  • Сферические фрезы
  • Грибковые фрезы (Lollipop)
  • Конические фрезы (только со сферическим наконечником)
