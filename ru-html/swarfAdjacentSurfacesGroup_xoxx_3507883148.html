Задает дополнительные поверхности рядом с обрабатываемыми для улучшения качества инструментального пути.

Стратегия может задевать эти смежные поверхности с тем же зазором, что указан для группы поверхностей.

Возможный результат

Улучшенное качество пути

Отключено
Потенциальный результат

Включено
Улучшено качество траектории инструмента.
