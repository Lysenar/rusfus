Определяет способ удаления оставшегося материала после чернового инструментального пути.

  • Получистовой проход - Выполняет один проход для одновременной очистки всех оставшихся выступов.
  • Инкрементальный проход' - Выполняет быстрый ход для очистки каждого гребня после каждого шага.
  • Получистовой проход' - Выполняет один проход для одновременной очистки всех оставшихся гребней.
