Определяет, как установить ориентацию инструмента.

Выберите Поворот и наклон, чтобы вручную указать углы поворота и наклона для инструмента.

Выберите Углы станка, чтобы указать углы станка для поворотных осей на основе кинематики станка, и увидеть модель станка на холсте для улучшенной визуализации.
