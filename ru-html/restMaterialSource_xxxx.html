Задает способ идентификации необработанных областей детали с остаточным припуском. Результирующий путьtargets только эти области.

  • Из предыдущих операций - Использует все пути в текущей настройке для идентификации оставшихся областей.
  • От инструмента - Позволяет определить инструмент и идентифицировать области, недоступные для него.
  • Из файла - Позволяет указать бинарный STL-файл, представляющий необработанные области.
  • Из тел - Позволяет указать твердотельные тела, представляющие необработанные области.
  • Из заготовки настройки - Использует заготовку, определенную в настройке, для представления необработанных областей.
