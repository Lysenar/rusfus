Отображается светло-синим цветом, задает опорное положение внешнего радиуса. Отсчет может вестись относительно Заготовки, Модели, заданного Радиуса, Диаметра или любого другого радиального положения. Это опорное положение может быть смещено с помощью положительного или отрицательного значения смещения.

Зазор - Задает внешний радиус относительно положения зазора.

Втягивание - Задает внешний радиус относительно положения втягивания.

НЗД (Наружный диаметр заготовки) - Задает внешний радиус относительно наружного диаметра определенной Заготовки.

НДМ (Наружный диаметр модели) - Задает внешний радиус относительно наружного диаметра определенной Модели.

Внешний радиус - Задает опорный внешний радиус относительно положения внешнего радиуса. Радиус зазора должен быть больше внешнего радиуса. Используйте параметр Смещение для необходимой корректировки.

Внутренний радиус - Задает внешний радиус относительно положения внутреннего радиуса. Радиус зазора должен быть больше внутреннего радиуса. Используйте параметр Смещение для необходимой корректировки.

ВДМ (Внутренний диаметр модели) - Задает внешний радиус относительно внутреннего диаметра Модели, как определено в Настройке. Используйте параметр Смещение для необходимой корректировки.

ВДЗ (Внутренний диаметр заготовки) - Задает внешний радиус относительно внутреннего диаметра Заготовки, как определено в Настройке. Используйте параметр Смещение для необходимой корректировки.

Выбор - Выберите любую грань, вершину или точку на модели для определения радиуса зазора. Используйте параметр Смещение для положительной или отрицательной корректировки по необходимости.

Радиус - Эта опция позволяет ввести значение радиуса в поле Смещение. Это значение отсчитывается от осевой линии детали и не учитывает ассоциативные изменения модели.

Диаметр - Эта опция позволяет ввести значение диаметра в поле Смещение. Это значение отсчитывается от осевой линии детали и не учитывает ассоциативные изменения модели.

Наибольший из... - Позволяет выбрать наружный или внутренний диаметр Заготовки или Модели и автоматически использовать вариант с большим диаметром.

Наименьший из... - Позволяет выбрать наружный или внутренний диаметр Заготовки или Модели и автоматически использовать вариант с меньшим диаметром.

Примечание: Внешний радиус должен быть меньше или равен радиусу втягивания, но больше внутреннего радиуса для генерации корректного инструментального пути.
