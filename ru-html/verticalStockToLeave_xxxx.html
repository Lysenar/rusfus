Параметр осевого припуска определяет количество материала, оставляемое в осевом (вдоль оси Z) направлении, то есть на торце инструмента.

Радиальный и осевой припуск

Указание положительного осевого припуска приводит к оставлению материала на пологих участках детали.

Для поверхностей, которые не являются строго горизонтальными, программа интерполирует между осевым и радиальным (стенка) значениями припуска, поэтому припуск, оставляемый в осевом направлении на этих поверхностях, может отличаться от указанного значения в зависимости от уклона поверхности и значения радиального припуска.

Изменение радиального припуска автоматически устанавливает осевой припуск на ту же величину, если вы не введете осевой припуск вручную.

Для чистовых операций значение по умолчанию составляет 0 мм/0 дюймов, то есть материал не оставляется.

Для черновых операций по умолчанию оставляется небольшое количество материала, которое затем может быть удалено одной или несколькими чистовыми операциями.

Отрицательный припуск

При использовании отрицательного припуска операция обработки удаляет больше материала из вашей заготовки, чем форма вашей модели. Это может использоваться для обработки электродов с искровым зазором, где размер искрового зазора равен отрицательному припуску.

Как радиальный, так и осевой припуск могут быть отрицательными числами, однако при использовании сферичной или радиусной фрезы с отрицательным радиальным припуском, который больше радиуса закругления, отрицательный осевой припуск должен быть меньше или равен радиусу закругления.
