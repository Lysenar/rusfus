Управляет количеством слоев проходов для построения элемента.

  • Весь элемент: Автоматически добавляет достаточно слоев для полного построения элемента.
  • Количество слоев: Позволяет указать количество слоев. Элемент может быть не полностью построен, если указано слишком мало слоев.
