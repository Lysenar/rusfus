Определяет способ задания ориентации инструмента.

  • Использовать WCS (по умолчанию) — использует систему координат заготовки (WCS) текущей задачи для ориентации инструмента.
  • Использовать Точку & Плоскость — задает конкретную ориентацию инструмента для этой операции из любой выбранной точки и плоскости в качестве начала координат и ориентации соответственно.
  • Использовать Систему Координат — задает конкретную ориентацию инструмента для этой операции из любого элемента системы координат в модели. Использует как начало координат, так и ориентацию элемента. Используйте, если в модели нет подходящей точки и плоскости для операции.
