Задает ориентацию поворотной оси в соответствии с установкой детали на станке.

Ориентация модели - Использует систему координат (СКД) текущей детали в качестве поворотной оси.

Ось X установки - Использует ось X системы координат заготовки (СКЗ) текущей установки в качестве поворотной оси.

Ось Y установки - Использует ось Y системы координат заготовки (СКЗ) текущей установки в качестве поворотной оси.

Ось Z установки - Использует ось Z системы координат заготовки (СКЗ) текущей установки в качестве поворотной оси.

Выбрать поворотную ось - Выберите плоскую грань, чтобы установить ось перпендикулярно ей, или выберите цилиндрическую грань, чтобы установить центр грани как точку, через которую проходит ось вращения.

Выбрать систему координат - Задает определенную ориентацию поворотной оси для этой операции из определенной пользовательской системы координат (ПСК) в модели. Использует как начало координат, так и ориентацию существующей системы. Используйте эту опцию, если в вашей модели нет подходящей точки и плоскости для операции.
