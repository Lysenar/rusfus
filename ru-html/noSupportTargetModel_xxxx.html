Определяет грани, тела или компоненты, которые не должны поддерживаться.

Щелкните отдельные грани тела. Двойной щелчок выбирает все тело. Указание исключаемых моделей может использоваться для создания двух разных действий поддержки на одном теле. Например, исключаемая модель для этой поддержки может затем быть поддерживаемой моделью для другой поддержки.

Выбранные грани

Выбранные грани не поддерживаются
