Определяет порядок отдельных операций.

Сохранить порядок — Обрабатывать features в порядке их выбора.

По операциям — Определяет порядок отдельных операций.

По инструментам — Определяет порядок операций по инструментам. Например, сначала обрабатываются все операции с плоской фрезой диаметром 3/4 дюйма, затем операции с другим инструментом, например, плоской фрезой 1/4 дюйма.
