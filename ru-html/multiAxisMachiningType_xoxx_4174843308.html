Определяет тип выполняемой обработки. Выберите опцию в зависимости от возможностей станка и геометрии детали.

  • 3-осевая - Для поверхностей, доступных без наклона инструмента. Поднутрения также возможны при использовании грибкового инструмента.
  • 4-осевая - Для элементов цилиндрических деталей.
  • 5-осевая - Для сложной геометрии, требующей наклона инструмента.
