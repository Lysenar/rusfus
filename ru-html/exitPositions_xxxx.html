Определяет позицию выхода инструмента для более безопасной обработки.

Выберите точку для определения позиции выхода. Точка должна быть как можно ближе к области, которую вы обрабатываете, например, к входу открытого кармана, чтобы точка была обнаружена во время расчета инструментального пути.

Если вы выбираете более одной точки, инструмент выходит из точки, к которой он ближе всего после завершения режущих движений.

В примере ниже обрабатывается открытый карман с подрезкой. Изначально инструмент втягивается в конце кармана, что вызывает столкновение с геометрией непосредственно над ним. Используя позицию выхода, вы можете заставить инструмент оставаться внизу и выйти из кармана, чтобы он мог втянуться с достаточным зазором.

Доступно для операций 2D Карман, 2D Контур и 2D Адаптивная.

Примечание: Инструмент выполняет перемещение без втягивания при движении к позиции выхода. Максимальное расстояние без втягивания должно быть достаточно большим, чтобы accommodate это перемещение. Для 2D Кармана и 2D Контура необходимо включить флажок Не втягивать инструмент, чтобы ввести Максимальное расстояние без втягивания.
