Указывает, какой метод используется для предотвращения блокировки.

Поступательное означает, что True Shape гарантирует отсутствие блокировки, перемещая компоненты вдоль осей X, Y и Z для проверки блокировки. Используя метод ограничительной рамки, True Shape упорядочивает компоненты на основе их ограничивающих рамок. Это гарантирует отсутствие блокировки без необходимости проверок, что приводит к более быстрому, но менее эффективному результату.
