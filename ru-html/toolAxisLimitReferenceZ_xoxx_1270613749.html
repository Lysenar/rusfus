Определяет reference при задании углов наклона инструмента.

  • Ось Z настройки - Задает углы наклона using the Z axis настройки.
  • Ось Z ориентации инструмента - Задает углы наклона using the Z axis ориентации инструмента.

Если ось Z вертикальна, то все horizontal areas относительно нее имеют inclination 0 градусов.

Совет: В scenarios, где вы меняете tool orientation для обработки flat areas, которые slightly inclined, можно использовать Setup Z для задания limits угла наклона, чтобы ограничить траекторию predictable regions.
