Тип соединения. Варианты: Полоса, Трапеция, Точки излома, Треугольники.

  • Полоса: Непрерывная лента, соединяющаяся с деталью по всей верхней кромке
  • Трапеция: Треугольники с усеченными вершинами
  • Точки излома: Как трапеция, но с дополнительным, перевернутым рядом трапеций сверху, образующим X-образные соединения
  • Треугольники: Треугольники, которые просто заканчиваются острыми вершинами, для очень деликатного крепления поддержек
