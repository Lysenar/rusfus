Отключает проверку на зарезы при выполнении переходных перемещений для ускорения расчёта траектории, особенно на больших моделях.

Проверка зарезов происходит между режущей частью инструмента и моделью.

Важно: Используйте эту опцию для ускорения программирования операции. После завершения настройки пересчитайте операцию ещё раз с отключённой опцией для обнаружения возможных зарезов.
