Редактор очков

1 - Принять для завершения создания полигона.
2. Нажмите «Отмена», чтобы отменить все изменения, внесенные с момента последнего принятия.
3. Нажмите «Удалить», чтобы удалить активный многоугольник.
4. Используйте режим вставки точки, чтобы добавить больше точек, щелкнув рядом с существующей линией или на ней. Вы можете нажать и удерживать, чтобы перетащить дополнительные точки, чтобы изменить форму многоугольника.
5. Используйте режим удаления точек, чтобы щелкнуть точки и удалить их.
6. Нажмите «Восстановить вид», чтобы вернуться к исходному виду, в котором вы находились, когда впервые начали размещать точки.
