Определяет, что использовать для смещения каждого прохода инструментального пути:

  • Граница обработки - Использует указанную вами Границу обработки для смещения инструментального пути.
  • Граница поверхности - Использует границу всех выбранных ведомых поверхностей для смещения инструментального пути.
  • Окружность в центре - Размещает небольшую окружность near центра поверхности и затем смещает эту окружность для создания инструментального пути.
  • Выбор кривых - Для типа Смешивание смещает две кривые до их встречи для создания инструментального пути. Если кривые параллельны друг другу, вы получаете инструментальный путь с более равномерно распределенными шагами. Для типа Гребень смещает одну или несколько кривых для создания инструментального пути.
  • Линия в центре - Размещает небольшую линию near центра поверхности и затем смещает эту линию для создания инструментального пути.
