Определяет, выбираются ли карманы вручную с помощью опции Выбранные карманы или автоматически, на основе их геометрии, с помощью опции Распознавание карманов.

Выбранные карманы - Используйте эту опцию при обработке детали с низкой сложностью, имеющей всего несколько карманов.

Распознавание карманов - Используйте эту опцию при обработке сложной детали со множеством карманов разного размера. Распознавание карманов экономит время, автоматически выбирая их на основе геометрии. Задайте ограничения геометрии в соответствии с геометрией вашего инструмента.

Примечание: При распознавании карманов учитываются только замкнутые карманы. Замкнутые карманы выбираются по их краям, а не по граням.
