Задает порядок операций в шаблоне.

Preserve order (Сохранять порядок) - Обрабатывает все операции в каждом экземпляре шаблона перед переходом к следующему.

Order by operation (По операциям) - Обрабатывает все вхождения каждой операции в шаблоне перед сменой операции.

Order by tool (По инструментам) - Обрабатывает все операции в шаблоне, использующие текущий инструмент, перед сменой инструмента.
