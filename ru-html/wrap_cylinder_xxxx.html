Выберите цилиндрическую поверхность, соответствующую геометрии, для которой нужно сгенерировать траекторию.

Обычно это обеспечивает точную траекторию, если стенки цилиндрического кармана совпадают с кромкой инструмента.

Если стенки кармана не совпадают с инструментом, выберите другой цилиндр обертки или调整 значение смещения для получения нужного реза.

При подстановке оси выбранная поверхность должна быть вдоль ротационной оси станка. Для многоосевой обработки поверхность может быть вдоль любой оси, если позиции траектории достижимы для станка.

Примечание: Поверхность не может быть конусом или irregular поверхностью.
