Выберите инструмент для обрезки:

Полигон - Позволяет размещать точки для создания полигона над областью траектории.
Плоскость — позволяет указать плоскость, покрывающую часть траектории инструмента.
Граница — позволяет выбрать замкнутый контур модели для использования в качестве границы области траектории инструмента.

Плоскость - Позволяет задать плоскость для покрытия области траектории.
Граница - Позволяет выбрать замкнутый контур на модели для использования в качестве границы над областью траектории.

Плоскости и границы ассоциативны с моделью, то есть могут перестраиваться при изменении модели.
