Задает максимальный угол наклона плоских участков, исключаемых из траектории.

На изображении ниже, указание угла Максимальный наклон больше, чем у самого крутого плоского участка, исключает все плоские участки из траектории.

Максимальный угол = 32 град

Минимальный наклон = 0 град
Максимальный наклон = 32 град

Примечание: Указание угла Минимальный наклон 0 означает, что траектория не генерируется на плоских участках с углом наклона 0, то есть на участках, лежащих в плоскости XY.
