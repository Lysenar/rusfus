Определяет способ задания ориентации инструмента:

Авто - Использует диапазон минимального и максимального углов наклона для определения ориентаций инструмента. Полезно для эффективного съема материала на сложных деталях.

Полуавто - Позволяет использовать комбинацию автоматического и ручного методов выбора. Полезно, если нужно сначала обработать в specificных ориентациях, а затем использовать диапазон углов наклона для остальных.

Выбор - Позволяет вручную задать ориентации инструмента для обработки детали. Используйте эту опцию, если можете оценить деталь и легко определить наиболее эффективные ориентации инструмента.
