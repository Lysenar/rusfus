Включите, чтобы выводить черновую обработку как canned cycle для вашего управляющего устройства.

Это может уменьшить объем кода, генерируемого для NC-программы. Это также ограничит некоторые advanced опции для траектории Profile Roughing, поскольку большинство canned cycles создают только basic boundary cut.

Примечание: Это может потребовать модификаций постпроцессора, специфичных для вашего управляющего устройства станка.
