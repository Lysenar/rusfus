Задает тип наклонного движения при выходе.

  • Зигзаг: Инструмент движется вперед-назад по зигзагообразной траектории при подъеме.
  • По профилю: Инструмент пытается следовать профилю обрабатываемой области.
  • По спирали: Инструмент движется по спирали при подъеме.
