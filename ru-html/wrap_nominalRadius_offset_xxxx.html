Смещает значение радиуса обертки. Смещение этого значения контролирует, сколько материала удаляется с края стенок. Обычно используется, когда стенки цилиндрического кармана не смоделированы точно для оберточной обработки.

При выполнении обертки траектория может быть точной только на одном радиусе - Радиусе обертки, который по умолчанию равен радиусу выбранного цилиндра обертки.

Значение смещения 0 использует радиус выбранного цилиндра обертки. На изображении ниже выбрана цилиндрическая поверхность внизу инструмента. В результате инструмент останавливается в конце нижней кромки.

На изображении ниже радиус обертки смещен на дюйм. В результате инструмент движется дальше в стенку.
