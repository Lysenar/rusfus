Определяет количество осей станка для использования при снятии фасок.

Как правило, использование multi-axis обработки позволяет достичь больше областей за одну операцию. Для 4- и 5-осевой обработки требуется станок с multi-axis возможностями.

  • 3-осевая обработка - Подходит для снятия фасок с призматических деталей, где кромки легко доступны инструменту.
  • 4-осевая обработка - Подходит для снятия фасок с цилиндрических деталей.
  • 5-осевая обработка - Подходит для снятия фасок со сложных деталей, требующих наклона инструмента для доступа к областям.
