Задает тип наклонного движения при входе.

  • Зигзаг: Инструмент движется вперед-назад по зигзагообразной траектории при погружении.
  • По профилю: Инструмент пытается следовать профилю обрабатываемой области.
  • По спирали: Инструмент движется по спирали при погружении.
