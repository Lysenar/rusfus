Задает максимальное количество раз, которое элементы сетки упрощаются вдали от области наплавления и в пределах платформы построения.

Введите значение от 0 до 6. Чем выше значение, тем грубее сетка. Значение 0 упрощает только верхний слой платформы построения. Упрощение — это объединение меньших элементов в областях с низкой детализацией. Если геометрия не подходит для упрощения, сетка не адаптируется.
