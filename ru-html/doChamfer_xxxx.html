Указывает, что операция контурной обработки будет использоваться для создания фаски.

Советы по выбору геометрии:

Острые углы

Кромки фасок

Острые углы - Выберите острый угол и задайте размер фаски с помощью параметра "Ширина фаски".

Кромки фасок - Выберите нижнюю кромку фаски. Ширина фаски рассчитывается автоматически.

Важно: Все выбранные кромки должны быть либо нижней кромкой фаски, либо нескошенными кромками. Если выбраны оба типа, то на уже существующих фасках будет создана фаска вдвое большего размера.
