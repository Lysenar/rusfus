Определяет направление генерации ступенчатых врезаний:

По оси инструмента - Размещает ступенчатые врезания вдоль оси инструмента. Полезно, когда между верхней и нижней направляющими кривыми есть прямой край. Для криволинейных поверхностей инструмент может не достигать определенных областей.

По линии контакта - Регулирует размещение ступенчатых врезаний с учетом режущей геометрии инструмента, например, конического инструмента, чтобы обеспечить постоянный контакт инструмента с поверхностью.

По касательной к поверхности - Размещает ступенчатые врезания, следуя кривизне поверхности. Ось инструмента может изменяться для каждого врезания, чтобы инструмент был касательным к поверхности. Полезно для обработки выпуклых поверхностей.
