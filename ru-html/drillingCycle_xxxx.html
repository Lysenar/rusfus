Выбирает тип сверлильного цикла для использования и отображает параметры для выбранной сверлильной операции. Показано с общепринятым G-кодом.

Сверление: Подача на глубину и быстрое втягивание - Для отверстий глубиной менее 3-х диаметров инструмента. (G81)

Зенкование: Подача на глубину, выдержка, быстрое втягивание - Выдержка используется для улучшения чистоты дна отверстия. (G82)

Стружколом: Использует несколько пеков с периодическим втягиванием инструмента для дробления стружки и/или подачи охлаждающей жидкости в отверстие. Для отверстий глубиной более трех диаметров инструмента. (G73)

Глубокое сверление: Аналогично стружколому, но полностью втягивается из отверстия для удаления стружки и/или затопления отверстия охлаждающей жидкостью. Также известно как пекование. (G83)

С выходом: Позволяет уменьшить подачу и скорость перед выходом из отверстия.

Направленное глубокое сверление: Пушечное сверло имеет одну режущую кромку. Направляющие подушки калибруют отверстие для получения прямого глубокого отверстия с точным диаметром.

Нарезание резьбы: Нарезает правую или левую резьбу в зависимости от выбранного инструмента.

Нарезание левой резьбы: Подача на глубину, реверс шпинделя, подача наружу - Вращается против часовой стрелки при входе в отверстие для нарезания резьбы. (G74)

Нарезание правой резьбы: Подача на глубину, реверс шпинделя, подача наружу - Вращается по часовой стрелке при входе в отверстие для нарезания резьбы. (G84)

Нарезание резьбы со стружколомом: Несколько раз подается внутрь и наружу, с каждым разом заглубляясь, пока не достигнет конечной глубины.

Развертывание: Подача на глубину и немедленная подача наружу по достижении конечной глубины. (G85)

Растачивание: Подача на глубину, выдержка, подача наружу. (G86)

Растачивание с остановкой: Подача на глубину, останов шпинделя, быстрое втягивание. (G87)

Тонкое растачивание: Подача на глубину, останов шпинделя, отвод от стенки, быстрое втягивание. (G76)

Обратное растачивание: Растачивание с обратной стороны существующего отверстия. (G77)

Фрезерование круглых карманов: Может использоваться для вызова пользовательского цикла фрезерования круглых карманов.

Расточное фрезерование: Может использоваться для вызова пользовательского цикла винтового расточного фрезерования.

Резьбофрезерование: Может использоваться для вызова пользовательского цикла винтового резьбофрезерования.

Пользовательский: Используется для вызова макроса инструмента датчика со станка. Требует пользовательский постпроцессор. См. расширенную функцию probing в выпадающем меню Настройка.
