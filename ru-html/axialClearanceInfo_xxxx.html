Выберите строку из таблицы выше, чтобы просмотреть ее значения осевого зазора.

При выборе View Absolute Values (Просмотр абсолютных значений) таблица обновляется для показа абсолютных значений, а здесь отображается начальное значение осевого зазора, заданное для этой группы.

При снятии выбора Показать абсолютные значения таблица показывает относительные значения, а здесь отображаются абсолютные значения.
