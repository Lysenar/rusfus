Задает подачу для перемещений, где инструмент не engaged в материал, но и не отведен.

Когда расстояние между резами больше Максимального расстояния без отвода, инструмент не будет retract на Высоту безопасного перемещения, а будет перемещаться с подачей между контурами реза. Этот параметр управляет подачей для этого движения.

Красная стрелка обозначает движение с подачей.
