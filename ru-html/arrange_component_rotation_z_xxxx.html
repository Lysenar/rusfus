Задает шаг, с которым этот компонент может поворачиваться по оси Z.

  • Global (Глобально): Поворачивается с шагом, заданным глобальной настройкой поворота.
  • 180 deg (180 град): Поворачивается с шагом 180 градусов.
  • 90 deg (90 град): Поворачивается с шагом 90 градусов.
  • Custom (Выборочно): Поворачивается с пользовательским шагом в градусах.
  • None (Нет): Без поворота по оси Z.
