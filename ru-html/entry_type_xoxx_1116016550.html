Определяет способ входа инструмента для начала режущего движения.

  • По нормали к поверхности - направление входа перпендикулярно поверхности модели. Полезно для обработки крутых и пологих областей, обеспечивая соответствующий вход для каждой зоны.
  • Вертикальная дуга - Инструмент входит по вертикали, со стороны оси инструмента. Обычно полезно только для обработки плоских областей.
  • Горизонтальная дуга - Инструмент входит по горизонтали. Направление дуги определяется автоматически.
  • Горизонтально влево - Вход выполняется по дуге влево перед началом режущего движения. Полезно для встречного фрезерования.
  • Горизонтально вправо - Вход выполняется по дуге вправо перед началом режущего движения. Полезно для попутного фрезерования.
  • Расширенное движение - Удлиняет движение входа в направлении, касательном к режущему движению.
  • Прямое - Размещает движение входа в плоскости XY. Эта опция позволяет задать Угол для управления направлением движения входа.
  • Наклонное врезание - Инструмент входит в заготовку с подачей для наклонного врезания, помогая удалить материал перед началом режущего движения.
