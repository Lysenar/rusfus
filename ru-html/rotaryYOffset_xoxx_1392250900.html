Смещает центр инструмента, чтобы резать кромками, а не центром.

Смещение инструмента

Диаметр инструмента = 0.3 дюйма
Смещение инструмента = 0 дюймов

Диаметр инструмента = 0.3 дюйма
Смещение инструмента = 0.14 дюйма

Смещение инструмента применяется в направлении, противоположном направлению вращения оси. При изменении направления вращения инструмент автоматически меняет позицию линейным движением без вращения для предотвращения damage.
