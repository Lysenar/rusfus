Задает точку крепления, используемую для соединения детали с точкой крепления стола.

Выбранная точка - Выберите точку на геометрии в области чертежа.

Точка габаритного box модели - Выберите точку на bounding box модели.

Точка габаритного box оснастки - Выберите точку на bounding box оснастки.

Точка габаритного box заготовки - Выберите точку на bounding box заготовки.

Примечание: Если выбрано Точка box оснастки, но оснастка не определена во вкладке Настройка, используется bounding box модели.
