Определяет способ размещения перемычек на траектории.

По контуру - Использует всю длину контура для размещения перемычек. По сравнению с опцией По сегменту перемычки распределяются более равномерно.

По контуру

Равномерно распределенные перемычки

По сегменту - Использует сегменты (участки траектории между двумя точками) для размещения перемычек. Если Расстояние между перемычками больше длины сегмента, то сегмент игнорируется и перемычка не размещается.

По сегменту

Использован один сегмент
