Обрабатывает выбранный Приоритет по Z-уровням. Один Z-уровень полностью обрабатывается перед переходом на следующий.

Если не выбрано, выбранная область Приоритета обрабатывается полностью перед переходом на следующую область.

Приоритет = Сначала верх
Сортировка по глубине откл.

Приоритет = Сначала верх
Сортировка по глубине вкл.

Приоритет = Сначала крутые
Сортировка по глубине откл.

Приоритет = Сначала крутые
Сортировка по глубине вкл.
