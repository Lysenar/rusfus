Задает направление для всех проходов.

Ручное направление прохода 0 градусов проходит параллельно оси x СКУ. Направление прохода 90 градусов проходит параллельно оси y СКУ.

Пример, когда может потребоваться использовать ручное направление прохода вместо автоматического - когда автоматическое направление приводит к проходам, которые упираются в стенку, заставляя инструмент касаться стенки multiple раз и оставляя нежелательные следы.
