Точка компенсации используется для ведения инструмента вдоль контуров, выбранных при создании траектории. Параметр Боковая компенсация определяет, с какой стороны инструмента находится точка компенсации.

Если точка компенсации не выбрана, Fusion выбирает ее автоматически. По вертикали точка компенсации находится на уровне острия инструмента (положение Z0). По горизонтали точка компенсации совпадает с максимальным диаметром инструмента. На изображении ниже точка компенсации представлена красной точкой.
