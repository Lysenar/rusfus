Определяет способ определения ориентаций инструмента:

Автоматический - Использует критерии поиска для автоматического определения наиболее подходящих ориентаций инструмента. Полезно для эффективного удаления материала на сложных деталях.

Полуавтоматический - Позволяет использовать комбинацию автоматического и ручного методов выбора. Полезно, если требуется сначала обработать в specific ориентациях before использования критериев поиска для automatic определения remaining ориентаций.

Выбор - Позволяет вручную specify ориентации инструмента для обработки детали. Используйте эту опцию, если можете оценить деталь и easily определить наиболее effective ориентации инструмента.
