Задает параметр поворота для каждого размещаемого компонента.

  • Global (Глобально): Использует глобальные настройки поворота для этого компонента.
  • Custom (Выборочно): Позволяет задать пользовательский поворот для этого компонента. Щелкните правой кнопкой мыши на имени компонента в таблице и выберите Edit (Правка), чтобы задать шаги поворота по осям X, Y и Z.
  • None (Нет): Сохраняет исходную ориентацию для этого компонента.
