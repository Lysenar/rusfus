Показано темно-синим цветом, этот параметр управляет самой внутренней позицией зоны резания. Для наружной обработки (OD) траектория инструмента будет резать до этой внутренней глубины. Для внутренней обработки (ID) это начальная высота внутренней зоны обработки.

Наружная обработка (OD).
Внутренний = Самая внутренняя зона обработки.

Наружная обработка (OD).
Исключение внутренних зон траектории.
Внутренний радиус регулирует глубину зоны резания.

Внутренняя обработка (ID).
Внешний = Самая внутренняя зона обработки.
Используется для ограничения зоны траектории.

Внутренняя обработка (ID)
Внешний радиус регулирует глубину резания.
Оставшийся материал показан желтым цветом.

Примечание: Внутренний радиус (Inner radius) должен быть меньше Внешнего радиуса (Outer radius) для наружной (OD) и внутренней (ID) работы.
