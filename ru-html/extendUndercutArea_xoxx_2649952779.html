Расширяет геометрию вокруг области подреза, в основном используется как мера предосторожности при обработке near any thin walls.

Изначально можно расширить области подреза при использовании опции Без подреза, чтобы оставить излишки материала вокруг undercut areas. Это обеспечивает большую жесткость around thin walls и позволяет обработать surrounding area larger tool.

Затем можно расширить области подреза на то же значение опцией Только подрезы, чтобы targeted excess material, позволяя использовать smaller tool при необходимости для undercut areas.
