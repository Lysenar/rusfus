Тип pattern (массива) для папки.

Linear (Линейный) - Создает массив УП/переходов по прямой линии или сетке.

Circular (Круговой) - Создает массив УП/переходов вокруг центра вращения.

Mirror (Зеркальный) - Создает зеркальные копии УП/переходов, отраженные относительно плоскости.

Duplication (По точкам) - Создает массив, используя УП/переходы как источник, а эскизные точки как цели.

Component (По компонентам) - Доступно только в сборках, позволяет копировать УП/переходы с одного компонента на другие в сборке.
