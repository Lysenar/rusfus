Определяет, как задается ориентация инструмента с использованием комбинации ориентации триады и опций начала координат.

Раскрывающееся меню Ориентация предоставляет следующие опции для установки ориентации осей триады X, Y и Z:

Ориентация по WCS настройки - Использует систему координат заготовки (WCS) текущей настройки для ориентации инструмента.

Ориентация по модели - Использует систему координат (WCS) текущей детали для ориентации инструмента.

Выбрать ось Z/плоскость и ось X - Выберите грань или ребро для определения оси Z и другую грань или ребро для определения оси X. Обе оси, Z и X, могут быть развернуты на 180 градусов.

Выбрать ось Z/плоскость и ось Y - Выберите грань или ребро для определения оси Z и другую грань или ребро для определения оси Y. Обе оси, Z и Y, могут быть развернуты на 180 градусов.

Выбрать оси X и Y - Выберите грань или ребро для определения оси X и другую грань или ребро для определения оси Y. Обе оси, X и Y, могут быть развернуты на 180 градусов.

Выбрать систему координат - Задает конкретную ориентацию инструмента для этой операции из определенной пользовательской системы координат (UCS) в модели. Использует как начало координат, так и ориентацию существующей системы координат. Используйте эту опцию, если ваша модель не содержит подходящей точки и плоскости для вашей операции.

Нормаль к поверхности и ось X (Расширение для механической обработки) - Щелкните в любом месте поверхности, чтобы установить ориентацию инструмента по нормали к этой поверхности, затем выберите грань или ребро для определения оси X.

Нормаль к поверхности и ось Y (Расширение для механической обработки) - Щелкните в любом месте поверхности, чтобы установить ориентацию инструмента по нормали к этой поверхности, затем выберите грань или ребро для определения оси Y.
