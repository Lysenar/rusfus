3-осевая - Полезна для обработки поверхностей, которые инструмент может легко достичь без наклона.

4-осевая - Полезна для обработки элементов в цилиндрических деталях.

5-осевая - Полезна для обработки сложной геометрии, требующей наклона инструмента для доступа к определенным областям, таким как подрезы.
