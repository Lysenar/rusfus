Задает величину уменьшения глубины Peck для каждого последующего Peck-прохода. Снижает нагрузку на сверло с увеличением глубины.

Множественные Peck-проходы Показано с уменьшением на 2мм

Пример: Глубина Peck 8мм, уменьшение глубины Peck 2мм и мин. глубина Peck 4мм. Первый проход: 8мм. Второй: 6мм. Третий: 4мм.

Значение 0.0 сохраняет одинаковую глубину Peck для всех проходов до достижения полной глубины.
