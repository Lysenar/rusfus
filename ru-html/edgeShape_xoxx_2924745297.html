Определяет объем удаляемого материала с ребра.

Снятие фасок:

  • Постоянная ширина - Расстояние между краями фаски одинаково для каждого ребра.
  • Постоянная высота - Глубина фаски одинакова для каждого ребра.

Скругление:

  • Постоянная ширина - Расстояние между концами скругления одинаково.
  • Постоянный радиус - Радиус скругления одинаков для каждого ребра.
