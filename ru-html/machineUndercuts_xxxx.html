Использует контурные проходы для обработки подрезов, иначе недоступных из-за нависающей геометрии. Подрезы можно обрабатывать на 3 или 5 осях.

Для 3-осевой обработки можно использовать инструменты типа "леденец", дисковые, бочкообразные или "ласточкин хвост".

Для 5-осевой обработки можно использовать более широкий range инструментов, включая шаровые фрезы. Кроме выбора Обрабатывать подрезы, используйте настройки вкладки Многоосевая для генерации 5-осевой траектории.

Обычно использование 5-осевого движения при обработке подрезов позволяет достичь больше областей за одну операцию, экономя время программирования.
