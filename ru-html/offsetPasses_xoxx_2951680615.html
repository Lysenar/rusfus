Смещает каждый последующий проход следующим способом:

Автосмещения - Автоматически определяет лучший способ смещения проходов.

Выбор - Смещает выбранные цепи из модели.

Самое длинное ребро - Пытается найти длинное ребро в геометрии для смещения. Для простой геометрии траектория может напоминать параллельный стиль.
