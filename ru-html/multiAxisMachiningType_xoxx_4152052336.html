Определяет тип выполняемой обработки.

Выберите опцию в зависимости от возможностей станка и геометрии детали. Некоторые опции отображают дополнительные настройки на вкладке Многоосевая.

  • 3-осевая - Для поверхностей, доступных без наклона инструмента.
  • 4-осевая - Для элементов цилиндрических деталей.
  • 5-осевая - Для сложной геометрии, требующей наклона инструмента.
