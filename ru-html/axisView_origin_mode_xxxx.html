Выпадающее меню Origin (Начало) предлагает следующие варианты расположения оси вращения:

Setup WCS origin (Начало WCS настройки) - Использует начало системы координат заготовки (WCS) текущей настройки для оси вращения.

Model origin (Начало модели) - Использует начало системы координат (WCS) текущей детали для оси вращения.

Selected point (Выбранная точка) - Позволяет выбрать вершину или ребро для оси вращения.
