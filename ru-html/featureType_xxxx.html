Определяет тип автоматически выбираемого элемента.

Только отверстия - Выбираются только отверстия в заданном диапазоне диаметров. Выступы игнорируются.
Только выступы - Выбираются только выступы в заданном диапазоне диаметров. Отверстия игнорируются.
Оба типа - Выбираются отверстия и выступы в заданном диапазоне диаметров.
