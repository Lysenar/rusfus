Выпадающее меню Начало координат предлагает следующие варианты расположения начала триады:

Начало СКД настройки - Использует начало системы координат детали (СКД) текущей настройки для начала инструмента.

Точка на поверхности (расширение обработки) - Использует выбранную точку на поверхности для начала триады.

Начало модели - Использует начало системы координат (СКД) текущей детали для начала инструмента.

Выбранная точка - Выберите вершину или грань для начала триады.

Точка ограничивающей рамки заготовки - Выберите точку на ограничивающей рамке заготовки для начала триады.

Точка ограничивающей рамки модели - Выберите точку на ограничивающей рамке модели для начала триады.

Примечание: Эта опция недоступна при использовании углов станка и когда модель станка видна на холсте.
