Показано светло-синим цветом, этот параметр управляет самой внешней позицией границы резания. Для наружной обработки (OD) траектория инструмента будет резать от этой внешней поверхности внутрь детали. Для внутренней обработки (ID) это лицевая сторона материала внутренней границы обработки. Подсказка: Щелкните левой кнопкой мыши на цветной полосе и перетащите.

Для наружной обработки (Outside Machining).

Самая внешняя граница обработки.
Внешний = Наружный диаметр заготовки (Stock OD) является обычным значением.

Траектория инструмента для наружной обработки.
Получившаяся обработанная граница.

Для внутренней обработки (Inside Machining).

Макс. глубина границы обработки.
Внешний = Наружный диаметр заготовки (Stock OD) является обычным значением.

Траектория инструмента для внутренней обработки
Получившаяся обработанная граница.

Примечание: Радиус отвода (Retract radius) должен быть больше Внешнего радиуса (Outer radius) для наружной работы (OD) и меньше Внутреннего радиуса (Inner radius) для внутренней работы (ID).
