Определяет конечную позицию траектории по X перед втягиванием по Z. Обычно полезно для траекторий на внутреннем диаметре детали.

Безопасное перемещение - Использует высоту безопасного перемещения для конечной позиции X.

Последняя точка траектории - Использует последнюю точку траектории для конечной позиции X. Это может улучшить эффективность траектории, так как инструмент не проходит дополнительное расстояние для завершения на высоте безопасного перемещения.
