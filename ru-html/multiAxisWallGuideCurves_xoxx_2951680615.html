Выберите геометрию из модели для указания в качестве направляющих кривых.

Для проходов типа Гребень можно выбрать одну или несколько направляющих кривых.

Для проходов типа Сопряжение сверху и снизу выберите только две направляющие кривые. Выберите одну направляющую кривую вверху стенки и одну внизу.
