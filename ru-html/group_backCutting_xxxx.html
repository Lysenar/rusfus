Для инструментов со специальной геометрией, где важен контроль утонения стружки. Активирует дополнительные элементы управления для установки скругления Радиуса входа и Дистанции выхода.

Доступно только когда тип цикла установлен на Обратное резание.

R = Радиус для входа обратного резания
D = Дистанция для снижения подачи

C = Направление резания - От патрона
R = Радиус для записи обратного резания
D = Расстояние для уменьшения подачи

Начиная траекторию ближе к стороне патрона, Radius обеспечивает плавный вход в рез и подачу в положительном направлении. На расстоянии до конца реза скорость подачи снижается, чтобы предотвратить сколы на детали или инструменте.
