Задает порядок резов:

Сначала глубина - Инструмент режет на полную глубину канавки перед смещением для следующего реза. Полезно для мягких материалов, где инструмент достаточно жесткий.

Сначала ширина - Инструмент снимает небольшой слой по глубине, затем смещается по ширине канавки, прежде чем перейти на следующую глубину. Полезно для твердых материалов для уменьшения прогиба инструмента. Может увеличить время цикла по сравнению с Сначала глубина.
