Используется на открытых контурах для удлинения начала и конца рассчитанной траектории инструмента. Создает линейное касательное удлинение на основе угла начальной и конечной точек. Это удлинение можно использовать в комбинации с Геометрия - Расстояние касательного удлинения (Tangential Extension Distance).

Примечание: Вы можете использовать опцию Контуры заготовки (Stock Contours), чтобы заставить траекторию выходить за пределы определенной Заготовки или выбранной границы. Отлично подходит для неправильных форм. Если вам нужно разное удлинение для каждого конца реза, вы можете использовать Вкладка Геометрия - Расстояние касательного удлинения (Tangential Extension Distance).
