Определяет геометрию для задания направляющих кривых. Проходы траектории смещаются от этих кривых.

Низ - Использует кривую низа стенки для смещения траектории.

Верх - Использует кривую верха стенки для смещения траектории.

Выбор - Позволяет выбрать направляющие кривые вручную.

Авто - Автоматически определяет наиболее подходящие направляющие кривые.
