Управляет подачей инструмента при подводе — движении инструмента при подходе к заготовке перед началом реза.

Для сокращения времени цикла рекомендуется устанавливать подачу подвода равной режущей подаче. Подачу подвода можно уменьшить для безопасных условий резания при большом глубине резания или малом зазоре на входе.
