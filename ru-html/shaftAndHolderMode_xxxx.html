Определяет, как траектория инструмента должна обрабатывать столкновение державки и/или хвостовика инструмента с заготовкой.

Отключено (Disabled)

Обрезка (Trimmed)

Отвод (Pull away)

Определить длину инструмента (Detect tool length)

Обрезка (Trimmed) - Участки траектории инструмента, которые приводят к нарушению безопасных расстояний между державкой и/или хвостовиком, обрезаются.

Отвод (Pull away) - Траектория инструмента отводится от заготовки для поддержания безопасного расстояния между державкой и/или хвостовиком.

Определить длину инструмента (Detect tool length) - Инструмент автоматически выдвигается дальше из державки для поддержания заданного безопасного расстояния между державкой и/или хвостовиком и заготовкой. Сообщение с указанием того, насколько инструмент выдвинут из державки, записывается в журнал.

Ошибка при столкновении (Fail on collision) - Расчет траектории инструмента прерывается, и при нарушении безопасного расстояния в журнал записывается сообщение об ошибке.
