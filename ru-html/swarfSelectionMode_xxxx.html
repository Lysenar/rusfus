Позволяет выбрать область обработки по непрерывным граням, отдельным граням или контурным парам цепей.

Ведущий режим = Поверхности
Режим выбора = Лица
Режим выбора = Грани
Выберите непрерывные поверхности.

Ведущий режим = Поверхности
Режим выбора = Ручной
Режим выбора = Вручную
Выберите отдельные поверхности.

Ведущий режим = Контуры
Режим выбора = Пары контуров
Выберите нижний и верхний контур
это будет управлять траекторией Swarf.
Режим выбора = Пары контуров
Выберите нижний и верхний контур,
которые будут вести инструмент.
Сначала выберите нижнюю цепь.
