Задает способ размещения перемычек.

  • По расстоянию - Позволяет указать Расстояние перемычки для равномерного размещения вокруг профиля. Используя меню Метод позиционирования, можно выбрать размещение вдоль контура или сегментов пути.
  • Количество перемычек - Позволяет указать Количество перемычек для равномерного размещения вокруг профиля.

Примечание: Перемычки размещаются перпендикулярно геометрии.
