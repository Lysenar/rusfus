Используется для станков с конфигурациями поворотных осей.

Когда отключено, размещение детали на физическом станке должно совпадать с размещением в Fusion.

Когда включено, выводится функция, позволяющая контроллеру станка переназначить программу, если деталь расположена не по центру относительно места, где она была запрограммирована в Fusion.

Примечание: Для использования этой функции требуется, чтобы система управления вашего станка имела функцию TCP. Свяжитесь с представителем станкостроителя для проверки этой возможности.
