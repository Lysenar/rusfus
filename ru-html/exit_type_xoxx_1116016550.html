Определяет способ выхода инструмента из режущего движения.

  • По нормали к поверхности - направление выхода перпендикулярно поверхности модели. Полезно для обработки крутых и пологих областей, обеспечивая соответствующий выход для каждой зоны.
  • Вертикальная дуга - Инструмент выходит по вертикали, в направлении оси инструмента. Обычно полезно только для обработки плоских областей.
  • Горизонтальная дуга - Инструмент выходит по горизонтали. Направление дуги определяется автоматически.
  • Горизонтально влево - Выход выполняется по дуге влево после завершения режущего движения. Полезно для встречного фрезерования.
  • Горизонтально вправо - Выход выполняется по дуге вправо после завершения режущего движения. Полезно для попутного фрезерования.
  • Расширенное движение - Удлиняет движение выхода в направлении, касательном к режущему движению.
  • Прямое - Размещает движение выхода в плоскости XY. Эта опция позволяет задать Угол для управления направлением движения выхода.
  • Наклонный выход - Инструмент использует наклонное движение для выхода из заготовки со скоростью подачи для наклонного движения.
