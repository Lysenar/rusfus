Задает тип выполняемой обработки.

Выберите опцию в зависимости от возможностей вашего станка и геометрии обрабатываемой детали. Некоторые опции отобразят дополнительные настройки на вкладке Многоосевая.

  • 3-осевая - Полезна для обработки поверхностей, которые инструмент может легко достичь без наклона.
  • Обмотка - Обматывает траекторию вокруг цилиндра для обработки замкнутых цилиндрических поверхностей. Полезна для обработки областей, недоступных при 3-осевой обработке. Требует станок с 4-осевой возможностью.
