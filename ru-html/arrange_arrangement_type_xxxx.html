Определяет способ компоновки выбранных компонентов для печати.

  • 2: Располагает компоненты на одном слое.
  • 3D Компоновка (Bounding Box): Располагает компоненты используя ограничивающую рамку.
  • 3: Располагает компоненты используя метод Монте-Карло.
  • 3D Компоновка (TrueShape): Располагает компоненты используя истинную форму.
