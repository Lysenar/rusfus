Открывает Библиотеку материалов заготовки, где можно выбрать материал заготовки.

Материал заготовки влияет на то, какие предустановки инструмента вы используете для обработки. При выборе инструментов для операций в этой настройке выбирайте из предустановок, связанных с выбранным материалом заготовки.
