Задает количество припуска, оставленного на стенках островков, для определения безопасности упорядочивания по островам.

Обычно значение Припуска на островках соответствует значению Радиального припуска операции, использованной для черновой обработки стенок.

Режущие проходы упорядочиваются по островам, если после учета припуска на соседнем островке остается минимальный зазор в диаметр инструмента.

Примечание: При недостаточном пространстве между островками для входа инструмента, режущие проходы по умолчанию упорядочиваются по глубине.
