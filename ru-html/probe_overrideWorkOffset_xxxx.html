Управляет системой координат для операции контроля.

Если выбрано, контроль выполняется в системе координат (Переопределение СКД), отличной от заданной в установке (Смещение СКД). Переопределение СКД для этой операции не обновляется, но Смещение СКД для последующих операций обновляется. Это фиксирует систему координат для контроля.

Пример обновленного Смещения СКД (1=G54), но неизменного Переопределения СКД (2=G55) для нескольких деталей:

Если не выбрано, контроль выполняется в той же системе координат, что и в установке (Смещение СКД). Смещение СКД для этой и последующих операций обновляется.
