Определяет способ расчета шага инструментального пути.

По поверхности - Использует поверхность модели для расчета шага. Может обнаруживать и обрабатывать подрезы.

От острия инструмента - Учитывает режущую геометрию инструмента при расчете шага. Полезно для обработки областей, где инструмент не может контактировать со всеми точками поверхности из-за своей формы. Например, области с внутренним радиусом меньше радиуса инструмента.

Использование этой опции позволяет использовать опцию Спираль для непрерывной обработки.

Примечание: От острия инструмента не подходит для обработки подрезов.
