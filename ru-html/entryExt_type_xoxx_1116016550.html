Определяет способ входа инструмента для начала режущего движения. Удлинение входа происходит перед движением входа.

  • По нормали к поверхности - Дуга перпендикулярна поверхности модели.
  • Вертикальная дуга - Удлиняет движение входа по вертикали, со стороны оси инструмента. Обычно полезно только для обработки плоских областей.
  • Горизонтальная дуга - Удлиняет движение входа по горизонтали. Направление дуги определяется автоматически.
  • Горизонтально влево - Удлинение выполняется по дуге влево перед выполнением движения входа.
  • Горизонтально вправо - Удлинение выполняется по дуге вправо перед выполнением движения входа.
  • Расширенное движение - Удлиняет движение входа по касательной.
  • Прямое - Размещает движение удлинения в плоскости XY. Эта опция позволяет задать Угол для управления направлением удлинения.

Изображения показывают различные типы удлинений входа, используемых после зигзагообразного наклонного движения входа.
