Расширяет выбор открытых цепей до границ ограничивающей рамки заготовки для определения замкнутого контура, который используется для расчета траектории инструмента для открытых карманов.

Когда расширенные открытые цепи пересекаются до достижения ограничивающей рамки, замкнутый контур не создается, и, следовательно, расчет траектории не может быть завершен.

В зависимости от геометрии открытого кармана выберите один из трех типов методов расширения:

Касательный (Tangent) - Расширяет все выбранные цепи по касательной.
Ближайшая граница (Closest boundary) - Расширяет все выбранные цепи в направлении, ближайшем к ограничивающей рамке.
Параллельный (Parallel) - Расширяет пересекающиеся выбранные цепи таким образом, что они становятся параллельными при достижении ограничивающей рамки. Оставшиеся цепи расширяются по касательной.
