Определяет тип переходного перемещения между проходами, находящимися рядом друг с другом в одной плоскости.

  • Без контакта (No contact) - Использует движение отвода для перехода между проходами. Дистанция отвода может зависеть от Политики отвода (Retraction Policy).
  • Прямая линия (Straight line) - Использует прямую линию для перехода между проходами.
  • Плавный (Smooth) - Использует плавные тангенциальные движения для перехода между проходами.
  • Следовать поверхности (Follow surface) - Связующее движение следует по обрабатываемой поверхности, фактически становясь режущим движением. Инструмент остается в контакте с поверхностью вместо отрыва от нее. Полезно в ситуациях, когда требуется небольшое связующее движение, чтобы избежать чрезмерного врезания инструмента.
