Показано темно-синим цветом, этот параметр управляет самой внутренней позицией границы резания. Для наружной обработки (OD) траектория инструмента будет резать до этой внутренней глубины. Для внутренней обработки (ID) это начальная высота внутренней зоны обработки. Подсказка: Щелкните левой кнопкой мыши на цветной полосе и перетащите.

Для наружной обработки (Outside Machining).

Самая внутренняя граница обработки.
Внутренний = Внутренний диаметр модели (Model ID) является обычным значением.

Траектория инструмента для наружной обработки.
Получившаяся обработанная граница.

Для внутренней обработки (Inside Machining).

Начальная высота границы обработки.
Внутренний = Внут. диаметр модели (Model ID) + радиус отверстия или Торцевая поверхность (Face).

Траектория инструмента для внутренней обработки
Получившаяся обработанная граница.

Примечание: Внутренний радиус (Inner radius) должен быть меньше Внешнего радиуса (Outer radius) для наружной (OD) и внутренней (ID) работы.
