Задает тип выполняемой обработки. Выберите опцию в зависимости от возможностей станка и геометрии детали.

  • 3-осевая - Полезна для обработки вертикальных кромок, особенно при криволинейном дне детали.
  • 4-осевая - Полезна для обработки элементов в цилиндрических деталях.
  • 5-осевая - Полезна для обработки сложной геометрии, требующей наклона инструмента для доступа к определенным областям.
