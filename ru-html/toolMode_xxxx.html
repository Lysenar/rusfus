Задает, в какое положение поворачивать револьдерную головку:

Инструмент для следующей операции - Поворачивает головку к инструменту, используемому для следующей операции в Браузере.

Выбрать инструмент - Поворачивает головку к инструменту, выбранному из библиотеки.

Пустая позиция инструмента - Поворачивает головку к указанной пустой позиции. Полезно для увеличения безопасного зазора при handling детали.
