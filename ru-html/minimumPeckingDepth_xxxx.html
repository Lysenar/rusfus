По мере уменьшения Глубины pecking на величину reduction, это значение задает minimum amount для сверления за один peck.

Показано с Минимальной глубиной 4мм

Пример: Глубина pecking 8мм, Уменьшение глубины pecking 2мм и Минимальная глубина pecking 4мм. Первый peck будет 8мм. Второй — 6мм. Третий — 4мм и все последующие — 4мм.
