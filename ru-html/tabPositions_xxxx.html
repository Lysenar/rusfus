В дополнение к автоматически генерируемым перемычкам, можно добавить manual перемычки для better фиксации детали во время обработки.

Щелкните в любом месте вдоль контура, чтобы добавить перемычки.

После добавления перемычки можно adjust ее позицию, перетаскивая ее:

Для удаления manual перемычки из canvas используйте Ctrl + Левый щелчок (Windows) или Cmd + Левый щелчок (Mac).

Подсказка: Если нужны только manual перемычки, в выпадающем меню Позиционирование перемычек выберите Количество перемычек и установите Перемычек на контур в 0.
