Определяет основной режим оси инструмента. В зависимости от геометрии модели выберите опцию, которая позволяет обработать инструментом наибольшее количество областей.

Основной режим для 4-осевой обработки:

  • На ось вращения - Острие инструмента всегда направлено на ось вращения.
  • На ось вращения с наклоном - Сначала острие инструмента направлено на ось вращения. Затем инструмент наклоняется в сторону оси вращения.
  • Нулевой угол опережения с наклоном - Сначала инструмент выравнивается по среднему углу между двумя кромками. Затем инструмент наклоняется в сторону оси вращения.

Основной режим для 5-осевой обработки:

  • Опережение - Наклоняет инструмент в направлении движения.
  • Наклон к Z - Наклоняет инструмент в любое место на конической границе вокруг оси Z ориентации инструмента.
