Отображается оранжевым цветом, задает опорное положение радиуса зазора. Отсчет может вестись относительно Заготовки, Модели, заданного Радиуса, Диаметра или любого другого радиального положения. Это опорное положение может быть смещено с помощью положительного или отрицательного значения смещения.

Втягивание - Задает радиус зазора относительно положения втягивания.

НЗД (Наружный диаметр заготовки) - Задает радиус зазора относительно наружного диаметра определенной Заготовки.

НДМ (Наружный диаметр модели) - Задает радиус зазора относительно наружного диаметра определенной Модели.

Внешний радиус - Задает радиус зазора относительно положения внешнего радиуса. Радиус зазора должен быть больше внешнего радиуса. Используйте параметр Смещение для необходимой корректировки.

Внутренний радиус - Задает радиус зазора относительно положения внутреннего радиуса. Радиус зазора должен быть больше внутреннего радиуса. Используйте параметр Смещение для необходимой корректировки.

ВДМ (Внутренний диаметр модели) - Задает радиус зазора относительно внутреннего диаметра Модели, как определено в Настройке. Используйте параметр Смещение для необходимой корректировки.

ВДЗ (Внутренний диаметр заготовки) - Задает радиус зазора относительно внутреннего диаметра Заготовки, как определено в Настройке. Используйте параметр Смещение для необходимой корректировки.

Выбор - Выберите любую грань, вершину или точку на модели для определения радиуса зазора. Используйте параметр Смещение для положительной или отрицательной корректировки по необходимости.

Радиус - Эта опция позволяет ввести значение радиуса в поле Смещение. Это значение отсчитывается от осевой линии детали и не учитывает ассоциативные изменения модели.

Диаметр - Эта опция позволяет ввести значение диаметра в поле Смещение. Это значение отсчитывается от осевой линии детали и не учитывает ассоциативные изменения модели.

Наибольший из... - Позволяет выбрать НЗД или НДМ и автоматически использовать вариант с большим диаметром.

Наименьший из... - Позволяет выбрать ВДЗ или ВДМ и автоматически использовать вариант с меньшим диаметром.

Примечание: Радиус зазора должен быть меньше радиуса втягивания, внешнего и внутреннего радиуса для генерации корректного инструментального пути.
