Определяет тип переходного движения между проходами, находящимися nearby на одной плоскости.

  • Без контакта - Использует движение отвода для перехода между проходами. Distance отвода может depend от Стратегия отвода.
  • Прямая линия - Использует прямую линию для перехода между проходами.
  • Плавный - Использует плавные tangential движения для перехода между проходами.
