Определяет, в каких направлениях генерируется растр. Штриховка по X и Y создает сетку, а Штриховка только по Y создает линии по Y.

Линии по X можно создать, используя Штриховка только по Y и установив Поворот по Z на 90 градусов. Стратегия заполнения недоступна, когда Тип заполнения — Полый.

Штриховка по X и Y (слева), Штриховка только по Y (в центре) и с отключенной опцией Заполнения (справа)
