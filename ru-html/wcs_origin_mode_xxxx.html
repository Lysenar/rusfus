Меню выбора начала координат предлагает следующие варианты расположения начала СКУ (WCS):

Начало модели - Использует начало системы координат текущей детали.

Выбранная точка - Выберите вершину или ребро для начала СКУ.

Точка ящика заготовки - Выберите точку на ограничивающем ящике заготовки.

Точка ящика модели - Выберите точку на ограничивающем ящике модели.

Из компоновки - Использует начало, определенное в настройке компоновки для Оболочки, для которой создана настройка. Для использования разверните узел Компоновки в браузере, щелкните правой кнопкой узел компоновки и выберите Создать настройки из компоновки.
