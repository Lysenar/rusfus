Открывает Библиотеку инструментов, где вы можете выбрать сопло для вашего типа работ.

Выберите подходящий Инструмент/Сопло для вашего процесса резки.

Подходящие типы Инструментов/Сопел
Гидроабразивная резка
Лазерная резка
Плазменная резка

Соответствующие типы инструментов/сопел
Водоструйная резка
Лазерная резка
Плазменная резка

Вы также можете выбрать значок
Новый гидроабразивный/плазменный/лазерный
для создания пользовательского Инструмента/Сопла.

Вы также можете выбрать
Новый водоструйный/плазменный/лазерный станок
значок для создания пользовательского инструмента/насадки.

Примечание: Тип выбранного Инструмента/Сопла повлияет на некоторые опции траектории.
