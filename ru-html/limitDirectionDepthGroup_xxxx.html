Ограничивает глубину обработки для каждой ориентации.

Часто используется для ограничения глубины обработки, чтобы не превысить длину режущей части инструмента. Также полезно для избежания достижения пределов осей станка.

Примечание: Это ограничение глубины доступно только при 5-осевой обработке. Для 3-осевой используйте верхнюю и нижнюю высоты для ограничения траектории.
