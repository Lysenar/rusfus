Управляет начальной позицией компонентов в процессе компоновки.

При включении компоненты располагаются с учетом их текущей позиции. Это полезно для точной настройки компоновки. При отключении компоненты располагаются заново без учета текущих позиций. Это полезно для completely новой компоновки.

Исходное расположение компонентов

Компоновка без учета текущих позиций
