Задает расстояние между деталью и поддержкой. Отрицательное значение заставит поддержки внедряться в mesh детали. При выборе пользовательского контура стержня это можно использовать в сочетании с Углом наверху для расчета площади контакта. Для расчета используйте формулу:

Расстояние до детали (отриц.) = (желаемый диаметр пятна - Ширина на конце) * (Ширина точки излома - Ширина на конце) / Высота на конце

Значение должно быть между -3 и 3 мм.
