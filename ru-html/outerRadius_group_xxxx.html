Отображается светло-синим цветом, управляет самой внешней позицией области резания. Для наружной обработки инструментальный путь будет резать от этой внешней поверхности внутрь детали. Для внутренней обработки это максимальная глубина внутренней области обработки.

Наружная обработка.
Внешний = Самая внешняя область обработки.

Наружная обработка.
Исключение внешних инструментальных путей для предварительно обработанных областей.
Внешний радиус регулирует торец области резания.

Внутренняя обработка.
Внешний = Самая внешняя область обработки.
Используется для ограничения области инструментальных путей.

Внутренняя обработка
Внешний радиус регулирует глубину резания.
Оставленный припуск показан желтым цветом.

Примечание: Радиус втягивания должен быть больше внешнего радиуса для наружной обработки и меньше внутреннего радиуса для внутренней обработки.
