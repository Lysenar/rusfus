При включении, выбор грани найдет все грани на той же плоскости, среди всех деталей в пределах границы заготовки.

Выберите любую грань на любой детали для сопоставления.

Примечание: Используется для фильтрации контуров по глубине грани. Используйте фильтр Петли для обнаружения всех внутренних и внешних контуров.
